## Step By Step

1. Cloning the Repo

```sh
    git clone https://github.com/WayanAndika/project-mvc.git
```

2. Install Modules

```sh
    npm install
```

3. Buat Database

```sh
    npx prisma generate
```

```sh
    npx prisma migrate dev
```

4. Jalankan Aplikasi

```sh
    node index
```

- Jika sudah install nodemon

```sh
    nodemon index
```
